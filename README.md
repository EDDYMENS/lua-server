# README

**Requirements**
- Lua 5.1 and above 
- Luasockets `luarocks install luasocket`

**Installation && Usage**
- `$ git clone https://github.com/EDDYMENS/lua-server.git`
- `$ lua server.lua app` NB: app is the entry lua script included in the repo, Type this out without the .lua extension
